# CASOS DE USO EXPANDIDOS

## UC1. Registrar Venda

### FLUXO PRINCIPAL
- [IN] Cliente informar sua identificação.
- [OUT] Sistema disponibiliza a opção de compra.
- [IN] Cliente informa o livro que deseja comprar.
- [OUT] Sistema exibe o valor total do livro.
- [IN] Cliente aceita o valor da compra.
- [OUT] Sistema apresenta as opções de pagamento.
- [IN] Cliente informa o meio de pagamento.
- [OUT] Sistema solicita a confirmação de pagamento.
- [IN] Cliente insere as informações e aceita o pagamento.
- [OUT] Sistema confirma o pagamento e exibe a tela de confirmação de compra.
